# 🚀 Space Invaders: Deluxe Edition (Versão Beta) v3.5

Bem-vindo à versão **Beta de Testes** do Space Invaders Deluxe Edition! 🧪

Esta é uma versão em desenvolvimento criada para testar novas mecânicas, ajustes de movimentação, escudos destrutíveis e batalhas contra chefões.

> 🔒 **ACESSO RESTRITO (BETA):** > Para jogar esta versão de testes, é necessário inserir a chave de acesso Beta na tela inicial.
> 
> 📢 **EM BREVE:** Assim que a fase de testes for concluída, o jogo será liberado **100% gratuito e aberto para todos**!

---

### 🎮 O que há de novo nesta versão:
- 🛡️ **Escudos Destrutíveis:** Barreiras de proteção que se desgastam com os tiros.
- 👾 **Batalha contra Chefão:** Boss com barra de vida a cada 3 fases.
- 📱 **Modo Mobile:** Suporte a botões sensíveis ao toque para jogar no celular.
- 🏆 **Sistema de Recordes:** Salva seus melhores placares com seu apelido.
- 💻 **Otimização para Notebooks:** Controles e cliques diretos sem travamentos.

# Space-Invaders-Beta-Edition
Space Invaders: Deluxe Edition (Beta) é uma reimaginação do clássico arcade! Enfrente ondas de alienígenas, proteja-se com escudos destrutíveis e vença chefões a cada 3 fases. Conta com suporte para notebook e celular, ranking Top 5 e chave de acesso exclusiva. Em breve a versão final será 100% gratuita para todos. Digite a chave e bom jogo!
